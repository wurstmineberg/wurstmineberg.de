The Wurstmineberg Minecraft Server

The Wurstmineberg Minecraft Server is whitelisted vanilla Minecraft server, and a subproject of the Wurstmannberg meta label, which doesn't really mean anything anymore. Come to think of it, it never really did. Oh well, now it does.

The server usually runs the latest version of Minecraft, which means we also try to run the latest stable snapshots. Therefore we can't really use awesome things like bukkit. We also don't run any serverside mods for that reason …Sorry.

If you're interested in playing with us, please see the IRC or Twitter section below. We're open, but fairly disorganized.

Overview map

We set up the Minecraft Overviewer to generate a daily overview map of our world.

It tends to drain a lot of resources, so we apologize for not running it every 10 minutes or so.
